## 3. Production Database Requirements

Document what is needed before connecting to live production DB:

- Exact production database name.
- Read-only MySQL/RDS user.
- SELECT-only permissions.
- No use of master/admin DB credentials.
- No write permissions.
- No schema permissions.
- No DROP/ALTER/INSERT/UPDATE/DELETE.
- Network/security group access requirements.
- Environment variables required.

Recommend using an expected-database guard per environment:

Development:
- `MYSQL_DATABASE=crbox_dev1`
- `EXPECTED_RDS_DATABASE=crbox_dev1`

Production:
- `MYSQL_DATABASE=<production_db_name>`
- `EXPECTED_RDS_DATABASE=<production_db_name>`

The app must abort if:
`SELECT DATABASE()` does not equal `EXPECTED_RDS_DATABASE`.
