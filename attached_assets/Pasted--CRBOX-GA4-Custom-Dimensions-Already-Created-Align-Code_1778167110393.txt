For context, the 24 GA4 custom dimensions have already been created manually inside the GA4 property.

You do not need to create new GA4 dimensions for this task.

This applies to the full GA4 analytics implementation, including #366 and #368.

CRBOX already has a centralized analytics architecture around:

- GTM

## Important Note About cta_text

cta_text is already registered in GA4, but it should be treated as secondary.

Prefer cta_id for reporting because it is cleaner and more stable.

If cta_text is sent, keep it normalized and controlled. Do not send dynamic or user-generated text.

---

## Important Note About High Cardinality

Avoid high-cardinality dimensions.

Do not send values that can create hundreds or thousands of unique values, such as:

full URLs
URL query strings
tracking numbers
invoice numbers
customer IDs
exact product names
exact package descriptions
exact chat messages
raw search queries
raw error messages
file names

Use buckets and categories instead.
